# Análise Hierárquica de Tarefas

## 1. Introdução

<p style="text-indent: 20px; text-align: justify">
Outro método empregado para realizar a análise de tarefas que o usuário possa buscar durante a utilização do UnB Idiomas, foi a Análise Hierarquica de tarefas (AHT).
</p>

## 2. O método AHT

<p style="text-indent: 20px; text-align: justify"> 
O método AHT consiste na identificação das tarefas que usuário podem necessitar executar utilizando a plataforma em análise pelos Designers, e ilustrar sua experiência no site até que o objetivo da tarefa esteja completo. Para que a análise ficasse mais intuitiva e compreensível o método foi empregado utilizando as representações:
</p>

-  Gráfica por meio de imagens de diagrama do fluxo do usuário pelo sistema, e legendas de apoio
-  Textual e descritiva das tarefas a serem realizadas

## 3. Motivo da escolha

<p style="text-indent: 20px; text-align: justify">
O método apresenta uma visão ampla acerca dos objetivos do usuário no sistema em análise, e por conseguinte auxilia os Designers na compreensão dos sentimentos, dificuldades, barreiras e necessidades em relação à aplicação como um todo. O método funciona muito bem para analisar o fluxo de usuário em tarefas relativamente mais complexas e que envolvem mais passos. 
</p>

## 4. A análise
<p style="text-indent: 20px; text-align: justify">
Abaixo podem ser visualizadas a análise de duas tarefas no sistema do UnB Idiomas, solicitar uma proposta de curso, a busca de informações sobre os valores de um curso.
</p>

## 4.1 Solicitar uma proposta de curso
 
!!! info "Objetivo"

    Realizar uma solicitação para um curso 

!!! note "Passos em ordem cronológica"
    <ol start="0">
        <li>Realizar uma solicitação para um curso</li>
        <li>
            Acesse o site do UnB Idiomas <br>
            1.1 Acesse a URL do site disponível em https://unbidiomas.unb.br    
        </li>
        <li>
            Navegue até a página de solicitação <br>
            2.1 Role a página até o final para visualizar os links no rodapé <br>
            2.2 Na seção "Propostas" clique em "Solicite sua proposta de curso"
        </li>
        <li>
            Acesse o formulário de proposta <br>
            3.1 Role a página para baixo até visualizar o botão "Solicite sua proposta" <br>
            3.2 Clique no botão "Solicite sua proposta" 
      4.0 Preencha o formulário de proposta
        </li>
        <li>
            Preencha o formulário de proposta <br>
            4.1 Preencha as informações no formulário de proposta <br>
            4.2 Clique no botão "Enviar"
        </li>
    </ol>
